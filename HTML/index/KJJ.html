WIN10快捷键:

CTRL + a 全选

CTRL + x 剪切

CTRL + c 复制

CTRL + v 粘贴

CTRL + z 撤销

CTRL + y 恢复

CTRL + s 保存

CTRL + shift + Esc 任务管理器

win + X + U + U 关机

win + X + U + R 重启

win + X + U + S 睡眠

win + X + U + I 注销

win + R 打开运行窗口

win + A 打开通知中心

win + W 打开windows link 工作区（可以用来临时绘图和截屏）

win + E 打开资源管理器（我的电脑或者我的文档）

win + I 打开设置

win + D 最小化所有窗口

win + L 锁屏

win + Shift + S 截图

win + TAB 任务总览（可以在这里增加新的桌面）

win + ↑ 最大化窗口或者打开最小化的窗口

win + ↓ 最小化窗口或者窗口化全屏的窗口

ALT +TAB 切换窗口

CTRL + win + ←或→ 切换不同的桌面

win + 数字键 打开对应的任务栏图标

shift + del 永久删除（无法恢复，不会到回收站！！！慎用）

AlT+F4 关闭窗口或者弹出一个电源选项
